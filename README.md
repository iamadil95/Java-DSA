## ğŸš€ Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code, or NetBeans)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/iamadil95/Java-DSA.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Java-DSA
   ```

3. **Open in your preferred IDE**
   - Import the project as a Java project
   - Ensure JDK is properly configured

4. **Run any Java file**
   ```bash
   javac <FileName>.java
   java <FileName>
   ```
